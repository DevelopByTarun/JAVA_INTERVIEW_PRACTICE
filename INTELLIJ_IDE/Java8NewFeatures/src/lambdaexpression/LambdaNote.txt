
LAMBDA EXPRESSION =>

THE EXPRESSION IS TO PROVIDE CLEAR AND CONCISE WAY TO REPRESENT ONE METHOD
INTERFACE USING AN EXPRESSION IS CALLED LAMBDA EXPRESSION.

ADVANTAGE =>

1. JAVA FIRST STEP INTO FUNCTIONAL PROGRAMMING

1. IT IS VERY USE FULL IN COLLECTION LIBRARY, IT HELP US TO ITERATE, FILTER COLLECTION.

2. IT IS USED TO PROVIDE IMPLEMENTATION OF INTERFACE AS FUNCTIONAL INTERFACE.
   
3. YOU DON'T NEED TO DEFINE METHOD AGAIN FOR ITS IMPLEMENTATION, SAVES LOTS OF CODE.

4. IT IS TREATED AS FUNCTION. SO COMPILER DON'T CREATE .CLASS FILE.
